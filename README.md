# streamlit-nav

Custom Streamlit navigation component to facilitate multipage applications 
developments. It is based on Nav component from rsuite.

## Getting started

Clone this repo:
```
git clone https://github.com/clabrugere/streamlit-nav.git
```

To run the component locally, first initialize and run the component template 
frontend:
```
$ cd streamlit-nav/navigation/frontend
$ npm install    # Install npm dependencies
$ npm run start  # Start the Webpack dev server
```

From a separate terminal, run the template's Streamlit app:
```
$ cd streamlit-nav
$ streamlit run navigation/__init__.py  # run the example
```
